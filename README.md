# Real State Marketplace

A real state NFT using ZoKrates zk-SNARKs for verification. 

Capstone project for Udacity Blockchain Nanodegree.

***

## Install

Install node dependencies

    npm install

***

## Tests

Run the tests with truffle

    truffle test

*** 

## Run the application

Run the truffle development node

    truffle develop

Deploy the contracts

    truffle(develop)> migrate

***

## Mint a token

Run zokrates using docker

    docker run -v <path_to_project>/zokrates/code:/home/zokrates/code -ti zokrates/zokrates /bin/bash

Move to the code/square directory

    cd code/square 

Compute a witness (a solution for a * a = b)

    zokrates compute-witness -a <a> <b>
    zokrates compute-witness -a 3 9

Generate the proof

    zokrates generate-proof

Run the mint script with a token ID

    truffle exec scripts/mint.js <tokenId> --network develop
